# Financial Playground

An interactive web app to explore how different financial decisions impact your net worth over time. Currently focused on comparing buying a house versus continuing to rent.

## Getting Started

Open `index.html` in a modern web browser. Adjust the inputs for income, expenses, housing costs and mortgage terms, then click **Plot** to visualize your projected net worth with and without purchasing a house.

## Development

This project uses [Chart.js](https://www.chartjs.org/) and [Tailwind CSS](https://tailwindcss.com/) via CDN links; no build step is required.

Run the test script (there are currently no tests):

```bash
npm test
```
